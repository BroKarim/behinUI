## Key Components

### Navigation Bar

- Clean, horizontal layout with responsive design
- Right-aligned menu items: "helo\_", "docs", "changelogs", "community"
- Top banner announcement with "$30M series B" funding news
- Subtle border-bottom separation for visual hierarchy

### Hero Content

- Two-column layout for larger screens, stacked on mobile
- Left column:
  - Label badge: "Powerful for developer."
  - Main heading: "Fast for everyone."
  - Subheading: "Most powerfull API built spesofically for the modern web."
  - CTA button: "Explore the doc" with arrow icon
- Right column :
  - Using Prism React Renderer for syntax highlighting
  - File indicator: "demo.tsx" with "animation.js"
  - Line numbers for better readability
  - Custom styling for TypeScript/React syntax
  - Integration logos with marquee animation

### Visual Design

- Background:

- Typography:
  - Modern monospace for code blocks
  - Sans-serif for main content
